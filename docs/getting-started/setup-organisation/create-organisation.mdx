# Create an organisation

The first time you log in to the No_Ops Portal, you are part of the `Playground` organisation.

`Playground` is a _sandbox_ account that is only used for demonstration purposes.

:::info
In order to connect your AWS account and create your own services, you have to set up your own organisation.
:::

## Create a new organisation

1. On the top left of the No_Ops portal, click on the arrow next to the `Playground` organisation, and select `Create an organisation`.

   <Image src="/docs/organisation/create_organisation.webp" />

2. Enter a name for your organisation and click `Add`.

   <Image src="/docs/organisation/organisation_name.webp" />

:::note[Next step]
You can now either [invite your team members](./invite-team-members.mdx) or [connect your AWS account](../connect-aws-account.mdx)!
:::
