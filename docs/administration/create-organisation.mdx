# Create an organisation

1. On the top left of the No_Ops portal, click on the arrow next to the `Playground` organisation, and select `Create an organisation`.

   <Image src="/docs/organisation/create_organisation.webp" />

2. Enter a name and code for your organisation, then click `Add`.

   <Image src="/docs/organisation/create_organisation_dialog.webp" />

:::note[Next step]
You can now either [invite your team members](./invite-team-members.mdx) or [connect your AWS account](../getting-started/connect-aws-account)!
:::
