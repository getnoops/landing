# Introduction

## Welcome to No_Ops!

No_Ops is the modern infrastructure platform for software engineers.

Rather than being slowed down by DevOps and worried about security and performance, No_Ops allows you to focus on your product.

## How to get started?

Start by [learning a few No_Ops concepts](docs/getting-started/concepts), then we will [connect your AWS account](docs/getting-started/connect-aws-account) to get you started.
